# Employee Management System (C++)

## Описание
Проект реализует простую систему учёта сотрудников на языке **C++**.  
Программа позволяет добавлять, редактировать, удалять и просматривать сотрудников.  
Для хранения информации используется динамический массив объектов `Employee`.

## Функционал
- Добавление нового сотрудника  
- Удаление сотрудника по ID  
- Редактирование данных сотрудника (ID, оклад, дата приёма, должность)  
- Просмотр списка сотрудников  

## Структура проекта
- **App** — основной класс приложения. Управляет меню, хранением и обработкой сотрудников.  
- **Employee** — класс, описывающий сотрудника. Содержит поля: ID, оклад, дату приёма, должность.  
- **Date** — класс для работы с датой. Хранит день, месяц, год. Позволяет вводить и выводить дату.  
- **EmployeeType** — перечисление должностей (лаборант, секретарь, менеджер, инженер, директор).  

## Использование
1. При запуске программы отображается меню.  
2. Пользователь может выбрать действие: добавить, редактировать, удалить или просмотреть сотрудников.  
3. Данные вводятся через консоль.  
4. После завершения работы вся динамически выделенная память освобождается.
